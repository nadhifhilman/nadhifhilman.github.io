Smart Door Lock RFID

Sistem Keamanan Pintu Berbasis Arduino & RFID

// Kode Arduino (dipersingkat tampilan, tetap full saat dibuka)
// Scroll di dalam box ini jika panjang

#include <Servo.h>
#include <SPI.h>
#include <MFRC522.h>
#include <LiquidCrystal_I2C.h>
#include <Wire.h>

Servo myservo;
MFRC522 mfrc522(10, 9);
LiquidCrystal_I2C lcd(0x27, 16, 2);

const int buzzer = 6;
const int LED1 = 5;
const int LED2 = 4;
const int LED3 = 3;
const int LED4 = 2;

void setup() {
  Serial.begin(9600);
  pinMode(buzzer, OUTPUT);
  pinMode(LED1, OUTPUT);
  pinMode(LED2, OUTPUT);
  pinMode(LED3, OUTPUT);
  pinMode(LED4, OUTPUT);

  SPI.begin();
  mfrc522.PCD_Init();

  myservo.attach(7);
  myservo.write(0);

  lcd.init();
  lcd.backlight();
  lcd.print("Smart Home Ready");
}

void loop() {
  if (mfrc522.PICC_IsNewCardPresent() && mfrc522.PICC_ReadCardSerial()) {
    if (mfrc522.uid.uidByte[0] == 198) {
      lcd.clear();
      lcd.print("Pintu Terbuka");
      myservo.write(135);
      delay(2000);
      myservo.write(0);
    } else {
      lcd.clear();
      lcd.print("Akses Ditolak");
    }
  }
}

Penjelasan Program

Program ini merupakan sistem Smart Door Lock menggunakan RFID. Arduino membaca kartu RFID dan mencocokkan UID kartu.

Jika kartu valid, maka servo akan membuka pintu, LED menyala, dan buzzer berbunyi. Jika tidak valid, maka sistem akan menolak akses.

Komponen utama:
- RFID RC522
- Servo Motor
- LCD I2C
- LED dan Buzzer
