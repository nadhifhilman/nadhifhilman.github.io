Penjelasan Script Automation Python

Langkah Penggunaan

1. Cari koordinat X dan Y posisi klik (Ketik python -m pyautogui diterminal)
2. Masukkan koordinat tersebut ke dalam kode program
3. Jalankan program dan siapkan WhatsApp Web / aplikasi chat / lainnya
4. Program akan klik, mengetik, dan mengirim pesan otomatis
5. Tekan tombol 'C' untuk menghentikan program

Kode Program

import pyautogui
import keyboard
import time

# GANTI koordinat sesuai hasil pencarian
x_pos = 1122 // contoh koordinat X
y_pos = 692 // contoh koordinat Y

print("automasi mulai dalam waktu 5 detik")
time.sleep(5) // Beri waktu 5 detik untuk mempersiapkan aplikasi chat

while not keyboard.is_pressed("c"): //Tekan tombol 'C' untuk menghentikan program
    pyautogui.click(x=x_pos, y=y_pos)

    pyautogui.typewrite("oke", interval=0.05) // 0.05 detik per karakter
    time.sleep(1)  // tunggu 1 detik

    pyautogui.press("enter")
